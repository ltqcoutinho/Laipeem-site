LAIPEEM – Landing Page (HTML puro simplificado)
===============================================

Esta versão NÃO envia dados. O botão de inscrição abre o Google Forms oficial em uma nova aba.

Como usar
- Substitua os arquivos de logo pelos seus PNGs com fundo transparente
  - laipeem-logo-full.png  (logo horizontal)
  - laipeem-selo.png       (selo/icone)
- Abra index.html localmente para testar.
- Publique no Linktree/Canva Sites/Notion colando o HTML inteiro ou hospedando o arquivo.

Dica Linktree:
- Use o bloco "Link" para apontar para um endereço onde o index.html está hospedado.
- Ou use um serviço simples de hospedagem estática (Netlify, Vercel, GitHub Pages) e cole o link no Linktree.

Qualquer ajuste de texto/cores, me peça que eu atualizo rapidinho :)
